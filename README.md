# 🌌 3D 태양계 시뮬레이션 (Solar System Simulation)

HTML, CSS, JavaScript 그리고 **Three.js**를 사용하여 구현한 인터랙티브 3D 태양계 시뮬레이션입니다.
웹 브라우저에서 태양과 행성들의 공전 및 자전 운동을 3D로 감상할 수 있습니다.

## ✨ 주요 기능

*   **3D 시각화**: 태양과 8개 행성(수성, 금성, 지구, 화성, 목성, 토성, 천왕성, 해왕성)을 3D 구체로 구현했습니다.
*   **리얼한 텍스처**: 각 행성의 특징을 살린 고해상도 텍스처를 적용했습니다. (천왕성, 해왕성은 색상으로 표현)
*   **애니메이션**:
    *   **공전 (Revolution)**: 태양을 중심으로 각기 다른 속도로 궤도를 돕니다.
    *   **자전 (Rotation)**: 각 행성이 스스로 회전합니다.
*   **사용자 컨트롤**:
    *   **재생/일시정지**: 시뮬레이션을 멈추거나 다시 시작할 수 있습니다.
    *   **카메라 조작**: 마우스 드래그로 회전, 휠로 줌인/줌아웃이 가능합니다.
    *   **설정 패널 (Settings)**: 각 행성의 자전 속도를 개별적으로 조절할 수 있습니다.
*   **편의 기능**:
    *   **행성 라벨**: 각 행성의 이름이 항상 표시됩니다.
    *   **방위표 (Compass)**: 우주 공간에서의 방향(N, S, E, W)을 확인할 수 있습니다.

## 🛠️ 기술 스택

*   **HTML5 / CSS3**: 구조 및 스타일링 (Glassmorphism UI 적용)
*   **JavaScript (ES6+)**: 시뮬레이션 로직
*   **Three.js**: 3D 그래픽 렌더링 및 카메라 제어

## 🚀 실행 방법

이 프로젝트는 텍스처 로딩을 위해 **로컬 웹 서버**가 필요합니다. (브라우저의 CORS 보안 정책 때문)

1.  저장소를 클론합니다.
    ```bash
    git clone https://github.com/redspy/solar_system.git
    cd solar_system
    ```

2.  로컬 서버를 실행합니다.
    *   **Python 3.x**:
        ```bash
        python3 -m http.server 8080
        ```
    *   **Node.js (http-server)**:
        ```bash
        npx http-server
        ```
    *   **VS Code**: 'Live Server' 확장 프로그램을 사용하면 편리합니다.

3.  브라우저에서 접속합니다.
    *   `http://localhost:8080`
